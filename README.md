# TweetBot
Logs into a Twitter account and tweets something.


Currently this bot logs into the specified account and tweets a random cat fact and exit.

INSTALLATION:

- npm init
- create a  .env
  - USERNNAME = <twitter username>
  - PASSWORD = <password>
- node index.js
